Detailed Description

Implements an HTML form

Author:
ESPIAU Nicolas <nicolas.espiau at fia-net.com>

void setAction(string $action) defines the action attribute of the form tag void setName(string $name) defines the attribute name of the form tag void setMethod(string $method) defines the attribute method of the form tag void setClass(string $class) defines the attribute class of the form tag void setId(string $id) defines the attribute id of the form tag void setAutosubmit(bool $autosubmit) sets the form to be submitted automatically or not
